# Cloud Service Models and AWS Fundamentals

This repository contains my learning and practice of cloud service models, AWS fundamentals, and cloud computing concepts.

## Cloud Service Models

Learned about the four major cloud service models:

### Infrastructure as a Service (IaaS)

- Provides virtual machines, storage, and networking.
- Users manage the operating system and applications.

### Platform as a Service (PaaS)

- Provides a complete platform for developing and deploying applications.
- Developers do not need to manage the underlying infrastructure.

### Software as a Service (SaaS)

- Software is delivered over the internet.
- No installation or maintenance is required by the user.

### Function as a Service (FaaS)

- Executes functions in a serverless environment.
- Infrastructure is managed automatically by the cloud provider.
