## Linux Permissions

Learned about:

- Owner, Group, and Other users
- User and group permissions:
  - Read (`r`)
  - Write (`w`)
  - Execute (`x`)
- Common permission sets (e.g., `755`, `644`)
- Managing permissions using `chmod`
- Changing file ownership using `chown`

## Web Servers

Learned the basics of servers and web servers.

Studied two commonly used web servers:

- Apache
- Nginx

## Common Server Errors

Learned about common web server errors and their causes:

- **404 Not Found** – The requested page or file does not exist.
- **403 Forbidden** – Access to the requested resource is denied.

## SSH Troubleshooting

Identified and resolved common SSH login issues.

Successfully established SSH connections between:

- Linux(virtual box) → Linux (Bootable)
- Linux (Bootable) → Linux (virtual box)

Learned how to troubleshoot authentication, network, and configuration-related SSH problems.
