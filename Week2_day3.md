# AWS Global Infrastructure and Cloud Services

This repository contains my learning and practice of AWS Global Infrastructure, cloud service models, AWS services, and cloud availability concepts.

## Cloud Service Models

Learned about the three major cloud service models:

### Infrastructure as a Service (IaaS)

- Provides virtual machines, storage, and networking.
- Users manage the operating system and applications.

### Platform as a Service (PaaS)

- Provides a platform for developing and deploying applications.
- Infrastructure is managed by the cloud provider.

### Software as a Service (SaaS)

- Software is delivered over the internet.
- Users can access applications without installation or maintenance.

Also learned the responsibilities of the **customer** and the **cloud provider** under each service model.
