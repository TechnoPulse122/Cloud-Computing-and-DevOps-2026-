## Cloud Service Models

### Infrastructure as a Service (IaaS)

- Provides virtual machines, storage, and networking
- Users manage the operating system and applications

### Platform as a Service (PaaS)

- Provides a complete development platform
- Developers can build and deploy applications without managing servers

### Software as a Service (SaaS)

- Software is delivered over the internet
- No installation or maintenance is required

### Function as a Service (FaaS)

- Executes functions in a serverless environment
- Infrastructure is managed by the cloud provider
