This repository contains my learning and hands-on practice with web servers, databases, and Apache web server configuration.

## Server Basics

Learned about:

- Difference between a Server and a Web Server
- How a browser requests a webpage from a web server
- Basic workflow of a web server

Explored different web servers:

- Apache
- Nginx
- Tomcat (Java)
- Kestrel (.NET)

Also learned about their basic configuration files.

## Databases

Got an introduction to:

- MySQL
- MongoDB
- Firebase

## Server-Side Rendering (SSR)

Learned the basics of Server-Side Rendering (SSR) and how web pages are generated on the server before being sent to the browser.

Performed the following tasks:

- Started the Apache service

- Restarted the Apache service without affecting the hosted website
