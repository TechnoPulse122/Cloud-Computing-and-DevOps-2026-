# Web Infrastructure and Server Management

This repository contains my learning and hands-on practice on web infrastructure, web security, FTP, and web server configuration.

## Web Infrastructure

Learned about:

- Domain names and their purpose
- DNS (Domain Name System)
- Name Servers
- Web Servers
- How a website is accessed from a browser

## Web Security

Learned the basics of HTTPS and SSL certificates.

- Difference between HTTP and HTTPS
- Role of SSL certificates in secure communication
- Importance of encrypted data transfer

## Advanced Search Techniques

Practiced using search operators such as:

- `site:`
- `filetype:`
- `intitle:`

Also learned how sitemaps help search engines index websites.

## FTP Configuration

Performed remote file management by:

- Creating an InfinityFree FTP account
- Connecting to the FTP server using WinSCP
- Uploading and managing website files

## Web Server Configuration

Installed and configured:

- Apache Web Server
- Nginx Web Server

Learned how to start, stop, and verify web server services using Linux commands.
