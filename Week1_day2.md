# Linux & Operating Systems

This repository contains my learning and practice of Operating Systems, Linux basics, and SSH configuration.

## Operating Systems

Learned about different operating systems:

- Linux
- Windows
- macOS
- Android
- iOS

Also learned the basic differences between them.

## Software Distribution Models

- **Open Source** – Source code is available.
- **Freeware** – Free to use but source code is not available.
- **Proprietary** – Paid or licensed software with restricted access.

## Linux Basics

### Linux File System

Learned about important directories:

- `/` – Root directory
- `/home` – User files
- `/etc` – Configuration files
- `/usr` – Applications and utilities

### Basic Linux Commands

Practiced the following commands:

```bash
pwd
ls
cd
mkdir
touch
cp
mv
rm
```

### Linux Users

Learned about:

- Root user
- Normal users
- Creating a new user using `adduser`
- Using `sudo`

### File Permissions

Understood:

- Read (`r`)
- Write (`w`)
- Execute (`x`)

Commands used:

```bash
chmod
chown
sudo
```
